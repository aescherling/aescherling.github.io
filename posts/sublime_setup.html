Setting up Sublime Text for data science

Adam Scherling

February 28, 2018

Sublime Text is a very powerful tool for writing code. It allows you to write and execute code for a variety of languages all in one place. It also provides a number of useful packages, keyboard shortcuts, and additional functionality that will make life a lot easier for you.

What follows is a brief guide to setting up Sublime Text to work as an IDE (integrated development environment).

Installation and basic setup

  1. Download + install: https://www.sublimetext.com

  2. Install package control: https://packagecontrol.io/installation

    This allows you to easily install packages for Sublime Text.

  3. Install sublimeREPL: https://sublimerepl.readthedocs.io/en/latest/

    A REPL (read-evaluate-print loop) is a command-line interface allowing you to execute code and view the results. The sublimeREPL package allows you to open up command lines within Sublime Text for a large number of languages (provided that you have the language installed on your machine).

  4. To open up a REPL for running code, open the command palette and type "sublimerepl" (or even just "repl" will work) and find the language you want to use.

    If you get an error message you might have to specify the path, e.g. https://github.com/wuub/SublimeREPL/issues/96

Keyboard shortcuts

Open the command palette:

  • Windows: ctrl+shift+p
  • Mac: cmd+shift+p

Split the screen:

  • Windows: alt+shift+n gives n screens
  • Mac: option+cmd+n gives n screens

Multiple cursors:

  • See https://www.sublimetext.com/docs/3/multiple_selection_with_the_keyboard.html

Custom shortcuts:

  • Go to Preferences -> Key Bindings. Enter the desired key bindings in the User window.
  • Example:

    [
  // Executes a selection of text in REPL
{ "keys": ["control+shift+j"], "command": "repl_transfer_current", "args": {"scope": "selection"}},

// Executes the entire file (build) in REPL
{ "keys": ["control+shift+k"], "command": "repl_transfer_current", "args": {"scope": "file"}},
   
 // Executes line(s) of text in REPL terminal
 { "keys": ["control+enter"], "command": "repl_transfer_current", "args": {"scope": "lines"}}
]
